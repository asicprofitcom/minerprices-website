# 🚀 Quick Start - Upload Miner Images

**System Status**: ✅ TESTED & WORKING

---

## 📸 In 3 Steps

### Step 1️⃣: Get Image URL
```
1. Go to: https://imgbb.com
2. Upload your miner photo
3. Copy image URL (e.g., https://ibb.co/abc123)
```
**Time**: 1 minute

### Step 2️⃣: Open Gallery Manager
```
Go to: https://minerprices.com/images.html
```
**Time**: Instant

### Step 3️⃣: Add Image
```
1. Search for miner name (e.g., "Antminer S21")
2. Click to select
3. Paste image URL
4. Click "Upload & Save"
```
**Time**: 1 minute

### Done! ✅
Image appears on miner page instantly.
